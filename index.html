Pengenalan Git dan Github

Git adalah sistem kontrol versi yang digunakan untuk melacak perubahan dalam kode sumber selama pengembangan perangkat lunak. Github adalah platform berbasis web yang menggunakan Git untuk menyimpan dan mengelola repositori kode.

Fungsi Git

  • Melacak perubahan dalam kode sumber
  • Membantu kolaborasi antar pengembang
  • Menyimpan riwayat versi kode
  • Memudahkan pengelolaan cabang (branching)
  • Memfasilitasi penggabungan (merging) kode
  • Memungkinkan rollback ke versi sebelumnya
  • Memudahkan pengelolaan konflik kode
  • Memberikan kemudahan dalam melakukan backup kode
  • Memungkinkan penggunaan di berbagai platform

    • Github

    Github adalah layanan hosting untuk proyek perangkat lunak yang menggunakan sistem kontrol versi Git. Github menyediakan antarmuka web untuk mengelola repositori Git, serta fitur tambahan seperti pelacakan masalah (issue tracking), wiki, dan halaman proyek.

    Fungsi Github

    • Menyimpan repositori kode secara online
    • Mendukung kolaborasi antar pengembang
    • Memfasilitasi pelacakan masalah dan bug
    • Menyediakan fitur wiki untuk dokumentasi proyek
    • Memberikan kemudahan dalam melakukan pull request
